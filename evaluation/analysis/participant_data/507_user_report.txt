This game is a memory based game. To carry on from where I left of the next player would need to know the names of the various characters involved in the game. The player would also need information of where each of these characters is located and the directions to get there. The game involves delivering potions to the charcaters as well as delivering messages from one character to another. The player would need infromation on what potion a charcater has requested and who is trying to deliver a message to whom.
